No more passwords or sensitive data sent in plain text over emails! Send it with a secure link, protected by a passphrase.

Your secrets are stored using advanced encryption and are only available for a limited amount of time. Shhh doesn't store your passphrases. They are only used to encrypt your secrets. Encryption method used: Fernet with password, random salt value and strong iteration count (100 000).

The project, built in Python (Flask) is open-source and available on Github, so feel free to contribute, star, fork, spread love! There is everything you need to host this app on your own private / local server if you want even more security / privacy.
